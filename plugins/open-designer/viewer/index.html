open-designer is a local design loop. Claude writes HTML drafts, you click elements to iterate – no cloud canvas, no backend.

  1. Ask Claude to design a page or component. Describe what you want.
  2. Drafts appear here automatically as Claude saves them.
  3. Toggle Select Elements, click what you want changed, type your feedback, paste the copied Markdown back into Claude.

Drafts live under .open-designer/drafts/<project>/. This tab auto-refreshes when files change.
