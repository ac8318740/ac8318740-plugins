open-designer is a local design loop. Claude writes HTML designs, you click elements to iterate – no cloud canvas, no backend.

  1. Ask Claude to design a page or component. Describe what you want.
  2. Designs appear here automatically as Claude saves them.
  3. Toggle Select Elements, click what you want changed, type your feedback, paste the copied Markdown back into Claude.

Designs live under .open-designer/designs/<name>/. This tab auto-refreshes when files change.

A design system is the tokens, voice, rules, and playable pages that every design reads from. Run /design-system in Claude Code to create one.
